Diamand Pattern Design.

This diamand pattern design was an attempt to get familiar with the loop feature of JAVA and it's scope. This diamand pattern is actually divided into 4 parts viz. Top left, top right, bottom left and bottom right. Code was dry run first to make sure that it's accurate and in 2 attempts I was able to run it properly.
